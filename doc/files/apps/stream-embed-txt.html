Stream Example (PIC32)

The programmer application is a self-contained firmware updater for the PIC32 USB Starter Kit II (http://www.microchip.com/DevelopmentTools/dm320003-2) and the I/O Expansion Board (http://www.microchip.com/DevelopmentTools/dm320002).

It’s source-code together with a MPLAB X project is located at apps\PIC32\stream.

Usage

This application uses the following connection between the PIC32 and the MGC3130

MGC3130PIC32Expansion Board
TS-LineRE8J10.37
Reset-LineRE6J10.6
I2CI2C3
-> SDASDA3/U1RXJ11.41
-> SCLSCL3/U1TXJ11.43

This configuration is the default one used in the example.  It could be changed by changing the DEV0_* defines in HardwareProfile.h or by providing an alternative to gestic_dev0_handler in gestic_custom.c.

The application uses the LEDs to signal incoming results.

LED 1 shows the state of the touch detection.  It turns on while touch is detected.

LED 2 is lit during ongoing communication.

LED 3 alternates when an gesture was detected.

This example uses the same IO-implementation as the Programmer Application (PIC32).
