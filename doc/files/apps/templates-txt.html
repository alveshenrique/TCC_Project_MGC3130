Windows Template

The Windows template for GestIC applications is an Visual C++ 2010 project for Win32 executables with the GestIC API related project configuration.  It is located at apps/Windows/template.

It uses the Dynamic API and links against gestic.lib and gestic.dll located at api/lib in the installation directory.
