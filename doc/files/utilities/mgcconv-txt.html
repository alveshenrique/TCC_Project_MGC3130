MGCConv

The MGC Image Conversion tool (MGCConv) allows extraction of firmware-images as C-structures from ENZ-Files.  Those C-Structures can be used with gestic_flash_image as demonstrated in Programmer Application (Windows/Linux) and Programmer Application (PIC32).

It is also suitable for checking C-Structures and ENZ-Files for errors.

Note

MGCConv is currently only available on Windows.  To extract an image for a different development platform like Linux the conversion has to be done on a Windows machine.  Alternatively one could use a Windows emulator or a compatibility layer like Wine.

Usage

mgcconv.exe <options> [<input> [<output>]]

Possible Options

--helpShows list of options
--input <file>Specifies the input file
--output <file>Specifies the output file
--checkDoes validation of <input> file
--what <image>Specifies which image to extract, image could be library or firmware
--name <varname>Specifies how to name the variable of the generated c-structure

Extracting C Structure

mgcconv.exe <input.enz> <output.c>

Instead of just extracting the contained C-structure this command will recreate for the contained image.  This could be used for extracting the image from damaged files that either contain wrong or no C-structure at all.

Loader images could be extracted with

mgcconv.exe -w loader <input.enz> <loader.c>

Validation of image files

mgcconv.exe -c <file>

This will write a list of potential problems found in the file like e.g. data inconsistency and missing or broken data blocks.

ENZ File Format

ENZ-Files are basically Zip-Files with the following structure

aurea.evcFile containing additional information that is used by the Aurea Software
content.jsonIndex file containing refernces to the other files together with meta-data
Library.encFirmware image in the ENC-format
Library.cFirmware image as a C-Structure
Library.settingsContaining the settings of the parameterization Aurea uses the content of this file to update Library.enc and Library.c
Loader.encBoot Loader image in the ENC-format
Loader.cBoot Loader image as a C-Structure
